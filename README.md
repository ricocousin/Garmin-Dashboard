# Fred's Feats — Training Dashboard

Personal training dashboard pulling activity data automatically from Garmin Connect, Apple Health, and the Anthropic API.

## Live Dashboard

[ricocousin.github.io/Garmin-Dashboard](https://ricocousin.github.io/Garmin-Dashboard)

## How it works

| File | Purpose |
|------|---------|
| `fetch_activities.py` | Fetches Garmin activities, LT data, generates AI coach summary |
| `receive_steps.yml` | Receives daily step data from Health Auto Export via Pipedream |
| `.github/workflows/fetch_activities.yml` | Runs fetch script daily at 6am Danish time |
| `index.html` | Dashboard, hosted via GitHub Pages |

**Daily (6am Danish time):** incremental fetch of new Garmin activities + AI coach summary
**First Sunday of each month:** full refresh of all historical Garmin data

## Data sources

### Garmin Connect (`runs.csv`, `strength.csv`, `summary.json`, `lactate.json`)
Fetched via the unofficial `garminconnect` Python library. Requires `GARMIN_EMAIL` and `GARMIN_PASSWORD` GitHub Secrets.

### Apple Health steps (`steps.csv`)
Automated daily pipeline:
1. **Health Auto Export** (iOS app, Premium) reads step count from Apple Health
2. Posts JSON to **Pipedream** webhook (`https://eoscoeru7yibehn.m.pipedream.net`)
3. Pipedream reformats payload and triggers GitHub `repository_dispatch`
4. `receive_steps.yml` workflow parses entries and appends to `steps.csv`

### AI Coach (`coach_summary.json`, `api_usage.json`)
Daily coaching summary generated by `claude-sonnet-4-6` via the Anthropic API. Requires `ANTHROPIC_API_KEY` GitHub Secret. Billed to a separate Anthropic API account (not the claude.ai subscription).

**Pricing note:** Token rates are hardcoded in `fetch_activities.py`. See the `# ── API pricing` comment block for current rates and last-verified date. Update manually if Anthropic changes pricing.

## GitHub Secrets required

| Secret | Purpose |
|--------|---------|
| `GARMIN_EMAIL` | Garmin Connect login email |
| `GARMIN_PASSWORD` | Garmin Connect password |
| `ANTHROPIC_API_KEY` | Anthropic API key for AI coach |

## Data files

| File | Description |
|------|-------------|
| `runs.csv` | All running activities with full metrics |
| `strength.csv` | All strength training sessions |
| `steps.csv` | Daily step counts from Apple Health |
| `summary.json` | Aggregated yearly/streak stats |
| `lactate.json` | Daily lactate threshold estimates (pace + HR) |
| `coach_summary.json` | AI-generated daily coaching summary + token usage |
| `api_usage.json` | Cumulative Anthropic API usage history |

## Dashboard sections

- **AI Coach** — data-anchored daily training summary with token cost display
- **Overview** — running and strength stats side by side
- **Intensity Minutes** — combined weekly training time averages
- **Activity Calendar** — monthly view with run/strength icons and weekly streak tracker
- **Daily Steps** — circular ring progress with 7-day history
- **Running** — personal bests, LT card with training zones, distance progress, weekly distance chart
- **Strength** — weekly sessions chart with monthly counts

## Stack

- Python + [garminconnect](https://github.com/cyberjunky/python-garminconnect)
- [Health Auto Export](https://healthyapps.dev) (iOS, Premium) + [Pipedream](https://pipedream.com) for steps
- Anthropic API (`claude-sonnet-4-6`) for AI coaching
- GitHub Actions (automation) + GitHub Pages (hosting)
- Chart.js + PapaParse (dashboard)
